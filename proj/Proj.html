Project 2: Small Video Game

Project 3: Hashtable Data Structures

For this I decided to write my own implementation of a hash table, I made this project using C++. The hash table was interesting to me as it provides both instant access to data but can also allow for multiple items in a single index of the hash table since I used chaining for when there is a collision on the the hash table. I build this hash table with the intent to store data about movies and T.V. shows and if there was two or more with the same hash key to store them in the same index of the hash table but add them to a linked list do avoid losing any data.
